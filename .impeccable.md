## Design Context

### Users
Developers del equipo que necesitan entender calculos de renta fija argentina. No son financieros — son ingenieros que necesitan implementar estas formulas correctamente. Usan la landing como referencia visual y documentacion viva.

### Brand Personality
**Didactico, Preciso, Confiable** — como un profesor que explica finanzas a ingenieros. Nunca condescendiente, siempre con el detalle tecnico accesible. Cada formula tiene su "por que".

### Aesthetic Direction
- **Visual tone**: Documentacion tecnica premium — inspirado en Stripe Docs, Vercel Docs, y Bloomberg Terminal (la densidad de datos, no el desorden).
- **Theme**: Dark mode principal (los devs lo prefieren), light mode como alternativa.
- **Typeface**: Inter (UI), JetBrains Mono (formulas, codigo, numeros).
- **Layout**: Narrow content (820px max), secciones colapsables, sticky nav lateral.
- **Color**: Indigo (#6366f1) como acento, verde para ganancias/positivos, rojo para perdidas/negativos, ambar para estimaciones/advertencias.
- **Anti-referencias**: PDFs del BCRA, planillas de Excel compartidas, documentacion sin formato.

### Design Principles
1. **Formula primero**: cada seccion arranca con la formula matematica renderizada, despues el ejemplo.
2. **Origen del dato visible**: siempre mostrar de donde viene cada variable (OMS, BCRA, CNV).
3. **Interactivo > estatico**: los devs aprenden tocando numeros, no leyendo parrafos.
4. **Progresivo**: de lo simple (LECAP) a lo complejo (BONTAM). Cada seccion construye sobre la anterior.
5. **Copy-paste ready**: el codigo y las formulas se pueden copiar directo.

### Quality Bar
**Flagship educativo** — la landing debe ser tan buena que los devs la usen como referencia diaria.

### Design Tokens
- Font: Inter (text), JetBrains Mono (code/numbers)
- Radius: 16px (cards), 8px (inputs/badges), 4px (inline code)
- Shadows: subtle (0 1px 2px), medium (0 4px 12px), elevated (0 8px 24px)
- Spacing: 4px base (4, 8, 12, 16, 24, 32, 48, 64)
- Max-width: 820px (content), 1200px (with sidebar)
- Transitions: 0.2s cubic-bezier(0.4, 0, 0.2, 1)
- Colors dark: bg #0f0f14, surface #1a1a24, border #2a2a3a, text #e4e4ef
- Colors accent: indigo #6366f1, green #22c55e, red #ef4444, amber #f59e0b
